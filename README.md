## Simple FastAPI

To initiate the project:

1. Create a virtual environment:
    - `$ python3 -m venv venv`
    - `source venv/bin/activate`

2. Install the dependencies:
    - `pip install -r requirements.txt`

To start locally add the follow command:
`$ uvicorn main:app --reload`
